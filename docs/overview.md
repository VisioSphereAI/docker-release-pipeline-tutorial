# Project Overview

This repository is a sample Flask application packaged for Docker with a simple CI workflow.

## What is included

- `app/` - Flask application package
- `tests/` - unit tests for the application
- `Dockerfile` - image build definition
- `docker-compose.yml` - local compose setup
- `.github/workflows/ci.yml` - CI pipeline for tests and Docker build
- `scripts/` - helper startup and test scripts
- `docs/` - documentation for setup and API usage
