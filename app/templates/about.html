About This Project 📚

🎯 Project Overview

This is an advanced Flask application demonstrating production-grade development practices with Docker containerization.

⚙️ Technologies Used
  • Python 3.12 - Core application language
  • Flask - Web framework with blueprints and factory pattern
  • Jinja2 - Template engine for multi-page support
  • Bootstrap 5 - Responsive UI framework
  • Docker & Docker Compose - Containerization
  • GitHub Actions - CI/CD pipeline
  • Gunicorn - Production WSGI server
📋 Feature List
Application Features
  • Multi-page HTML templates
  • Interactive calendar system
  • Task management system
  • Static CSS/JS assets
  • Responsive design
Development Features
  • App factory pattern
  • Blueprint-based routing
  • Error handling
  • Environment configuration
  • Unit tests with pytest
